# 山寨币的价格

[Index](index.md)

比特币是所有其他加密货币的共同参照物，我们可以以它来估算其他加密货币的价格。估算依据：

- 山寨币的总价值占比特币总价值的比例
- 山寨币的发行量，包含已发行的，计划中的全部发行量
- 比特币的当前价格

一般说，山寨币的价值不可能超过比特币，它能稳定在比特币的多少分之一即是合理的。比特币是通用货币，汇聚了最多的共识，因此它的价值最大。山寨币往往针对某个细分领域，它的价值自然只能比比特币小。山寨币的极限价格就是山寨币价值和比特币价值相等之时，山寨币的价格越接近这个极限越不合理。

- $value = price \times supply$
  - 如果了解山寨币的所能代表的市值，可以用这个公式直接估算山寨币的价格
- $value_{altercoin} < value_{bitcoin}$
- $price_{altercoin} < \dfrac{value_{bitcoin}}{supply_{altercoin}}$
  - 可以用这个公式评估以比特币为参照的价格是否合理

## 如何大致判断一个币的价格是否合理？

- 原理：任何币的总价值都不可能超过比特币
- 根据发行量和价格计算总价，然后和比特币的总价比较，如果超过了，肯定不靠谱，可以买跌。如果达到不合理的比例也肯定不靠谱，买跌！
- 比如 Filecoin，发行量是20亿，它的量是比特币的100倍，它的价格顶天了也不会超过比特币的1/100。它只是一个细分市场，体量能达到比特币的十分之一都不可能，所以，它的合理比价可能不会超过比特币的1/1000，按现价就是70美金。
- 合理币价 = 比特币价格 * (币的总量/比特币总量) * 币的价值总量相对比特币的比例
